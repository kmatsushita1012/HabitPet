# Habit Pet 企画書（ドラフト）

## タイトル

Habit Pet（仮）

> 悪習慣を代わりに受ける、かわいい生き物。

## 概要

禁煙・禁酒・禁ギャンブルなどの悪習慣を、かわいいキャラクターの状態変化として可視化するiOSウィジェットアプリ。

ユーザーの行動に応じてキャラクターの「体調・空気状態」が変化し、ホーム画面上で常に視認されることで行動改善を促す。

## 対応モード（初期）

- 🚬 禁煙モード
- 🍺 禁酒モード
- 🎰 禁ギャンブルモード

※すべて同一キャラクターに反映し、習慣ごとに切り替え可能。

## コア体験

1. ユーザーが悪習慣を入力（＋ボタン）
2. キャラクターが即座に苦しむ
3. 継続すると徐々に回復

> 数値ではなく「感情」で理解する体験。

## UX設計

### ① ウィジェット（メイン体験）

- サイズ: iOS 2×2（最重要導線）

#### レイアウト

```text
┌──────────┐
│ キャラ（状態表示） │
│                    │
│──────────│
│ 状態テキスト [+] │
└──────────┘
```

#### 要素

- キャラ（主役）
- 5段階状態変化
- 常時表示
- 状態テキスト（1行）
  - 例: 「3日連続」「今日は2本」「悪化中」
- ＋ボタン
  - ワンタップで記録

#### 取り消しUX

- 入力後5秒間のみ表示
- 「1本吸った」→「取り消す（5秒）」
- 5秒後に自動消滅
- 直前の1操作のみ取り消し可能
- 状態テキストの位置に代わって表示

#### フィードバック

- 入力直後にキャラクターが苦しむ
- 煙や表情変化を表示

> 操作 → 即感情反応

### ② アプリ本体

#### 構造

- 1画面構成
- 横スクロールで習慣を切り替え（ページング）

#### 各画面構成

- キャラ（大）
- 状態テキスト
- [＋入力] [取り消し]
- 簡易履歴

#### ナビゲーション

- 右上「＋」で習慣追加
- 横スワイプで習慣切替

#### 履歴

- 詳細履歴はアプリ内のみ表示
- ウィジェットには表示しない

## 習慣設定

### 初期設定

- 基準値（baseline）
- 「先週どれくらい？」を入力
- または初期入力から推定

### 設定項目

- カウント方式
- 通し
- 日次リセット
- 目標
  - ○日継続
  - 任意終了日

## 状態ロジック

### baseline（基準）

- 過去7日平均を基準とする

### Lv（5段階）

- usage / baseline に応じて状態が変化

### 挙動

- 悪化: 即時反映（＋入力で即悪化）
- 回復: 7日平均に基づき徐々に回復
    - 重み付き平均で直近の値が反映されやすく

> UXルール: 悪化は速く、回復は遅い。

## キャラクター設計

### コンセプト

- かわいい生き物（例: ハムスター）
- 丸くてシンプル
- ミニマル背景（白）

### 状態（5段階）

1. 元気（クリーン）
2. 少し不調
3. 苦しそう
4. 重度不調
5. 危険（酸素マスク）

## 表現

- 表情変化
- 呼吸変化
- 煙・空気エフェクト

## アニメーション

- 呼吸（常時）
- 咳（中〜重度）
- 震え（重度）
- 煙（入力時）

## マネタイズ

### 無料

- 1習慣
- 5キャラ

### Pro（サブスク / 買い切り）

- 複数習慣
- キャラ追加
- カスタマイズ（色・テーマ）

## 差別化

| 項目 | 既存 | 本アプリ |
| --- | --- | --- |
| UI | 数値・グラフ | キャラ |
| 動機 | 理性 | 感情 |

| 習慣 | 個別管理 | 体験統合 |

## MVP（最小構成）

- 禁煙モードのみ
- 1キャラ（ハムスター）
- 5段階状態
- 2×2ウィジェット
- ＋入力＋取り消し
