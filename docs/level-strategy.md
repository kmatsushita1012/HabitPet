# キャラクターレベル算出戦略（MVP・再改訂版）

## 1. 目的
禁煙・禁酒の記録から、キャラクターの体調を5段階で判定する。

## 2. 入力
- `x`: その日の実績（本数 / 杯数）
- `g`: その日の目標値

## 3. 1日スコア
スコアは高いほど悪い。

### 3.1 目標がある場合（`g > 0`）

| 条件 | dayScore |
| --- | --- |
| `x <= g` | `0` |
| `g < x < 2g` | `2` |
| `x >= 2g` | `3` |

### 3.2 目標が0の場合（`g == 0`）

| 条件 | dayScore |
| --- | --- |
| `x == 0` | `0` |
| `x == 1` | `1.5` |
| `x == 2` | `2.5` |
| `x >= 3` | `3` |

## 4. 週間スコア
直近7日間の重み付き平均で算出する。

| 日 | 重み |
| --- | --- |
| 今日 | `1.0` |
| 昨日 | `0.8` |
| 2日前 | `0.6` |
| 3日前 | `0.5` |
| 4日前 | `0.4` |
| 5日前 | `0.3` |
| 6日前 | `0.2` |

```text
weeklyScore = Σ(dayScore × weight) / Σ(weight)
```

- 重み合計: `3.8`

## 5. キャラクター体調（5段階）
- `weeklyScore == 0.00`: `Lv1`（元気）
- `0.00 < weeklyScore <= 0.70`: `Lv2`
- `0.70 < weeklyScore <= 1.40`: `Lv3`
- `1.40 < weeklyScore <= 2.10`: `Lv4`
- `2.10 < weeklyScore <= 3.00`: `Lv5`（最悪）

## 6. 設計意図
- 目標達成日は悪化させない
- 目標超過から明確に悪化させる
- 2倍以上の超過は強く悪化させる
- 直近7日で傾向を見ることで、今日だけで決まりすぎないようにする
